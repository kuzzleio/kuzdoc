# Usage

<!-- usage -->
```sh-session
$ npm install -g kuzdoc
$ kuzdoc COMMAND
running command...
$ kuzdoc (-v|--version|version)
kuzdoc/2.0.0 darwin-x64 node-v12.18.3
$ kuzdoc --help [COMMAND]
USAGE
  $ kuzdoc COMMAND
...
```

## `kuzdoc help [COMMAND]`

display help for kuzdoc

```
USAGE
  $ kuzdoc help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
